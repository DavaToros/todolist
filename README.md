# Todo API

Простой REST API для управления задачами (список дел), написанный на **FastAPI** с использованием **SQLite3** и **Docker**.  
Подходит для демонстрации базовых навыков бэкенд-разработки: проектирование REST API, работа с БД, контейнеризация.

## Возможности

- Создание, чтение, обновление и удаление задач (полный CRUD)
- Постоянное хранение данных в SQLite3
- Интерактивная документация API (Swagger UI и ReDoc)
- Готовность к запуску в Docker

## Технологии

- **FastAPI** – современный веб-фреймворк на Python
- **SQLite3** – лёгкая встраиваемая база данных
- **Docker** и **Docker Compose** – контейнеризация и оркестрация
- **Uvicorn** – ASGI-сервер
